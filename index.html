Holt-Winters triple exponential smoothing · 80% prediction interval

Holt-Winters Triple Exponential Smoothing

Additive seasonal model decomposing daily caffeine into level, trend, and day-of-week seasonal components, updated recursively as new data arrives.

Level: L[t] = α(y[t] − S[t−m]) + (1−α)(L[t−1] + T[t−1])
Trend: T[t] = β(L[t] − L[t−1]) + (1−β)T[t−1]
Seasonal: S[t] = γ(y[t] − L[t]) + (1−γ)S[t−m]
Forecast: ŷ[t+h] = L[t] + h·T[t] + S[t−m + (h mod m)]

m = 7 (weekly)    α = 0.3 (level)    β = 0.1 (trend)    γ = 0.2 (seasonal)
80% PI: ŷ ± 1.28 · RMSE · √h
Requires ≥14 days of history — falls back to double exponential smoothing otherwise.
